# G32_DC_PA-26.1

Projeto em React + JavaScript para demonstrar o algoritmo de dividir para conquistar aplicado ao problema de par de pontos mais próximos.

## Como executar

1. Instale as dependências com npm install.
2. Inicie o projeto com npm run dev.

## O que esta versão inicial entrega

- Seleção de modo: multiplayer local ou contra bot.
- Tabuleiro 10x10 para posicionamento de 5 pontos por jogador.
- Resolução da rodada com ponto neutro e comparação de proximidade.
- Destaque visual do par de pontos mais próximo usando divide and conquer.
